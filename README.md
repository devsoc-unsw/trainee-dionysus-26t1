# trainee-dionysus-26t1

## Getting Started

Hello dionysus disciples first you must install bun:

```sh
curl -fsSL https://bun.sh/install | bash
```

You will need to reload your shell's config file for it to work. 

## Backend

Go into the backend and install all the bun packages:

```sh
cd backend
bun install
```

And to start the backend server, run:

```sh
bun run server.ts
```

Backend runs on:

```
http://localhost:8080
```

## Frontend

Go into the frontend and install all the bun packages:

```sh
cd frontend
bun install
```

And to start the frontend server, run:

```sh
bun run dev
```

Frontend runs on:

```txt
http://localhost:5173
```
